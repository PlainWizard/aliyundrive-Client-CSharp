# aliyundrive-Client-CSharp
C# Client for aliyundrive

## 概述

由于需要找一个小网盘来放e小天应用,找了一堆也没合适的
看了下阿里云盘在公测不错,而且全开放的接口形式,非常适合拿来做小网盘,1T空间足够用了,所以写了个c#同步文件到网盘的客户端

## 运行环境
- .NET Framework 4.6
